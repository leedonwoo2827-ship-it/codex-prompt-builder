# 🎨 프롬프트 빌더

이미지(카드뉴스·PPT·표지 등) 만들 때 쓰는 **프롬프트를 대신 만들어 주는 사내 도구**입니다.
원고 파일을 넣고 버튼 몇 개만 고르면, 이미지 스튜디오에 바로 붙일 프롬프트가 나옵니다.
설치·실행은 **파일 더블클릭**만 하면 됩니다. (개발 지식 필요 없음)

---

## 처음 한 번만 (설치)

1. `setup.bat` **더블클릭** — 잠시 기다리면 설치가 끝납니다.
2. 끝나면 `run.bat` **더블클릭** — 브라우저가 자동으로 열립니다.
3. 처음 열리면 **설정 화면**이 나옵니다.
   - **프로바이더** 선택(회사 LiteLLM / 샤오미 MiMo)
   - **API 키** 입력 → **「연결 확인」** 눌러 ✅ 확인 → **「저장」**
   - (키는 담당자에게 받으세요)

> 다음부터는 `run.bat`만 더블클릭하면 됩니다.

---

## 사용법 (화면 3개)

### 1) 프롬프트 생성기 — 가장 쉬움 ⭐
1. **원고 파일**(PDF·한글 등)을 끌어다 놓기
2. **용도 · 톤 · 색 · 스타일 · 컷 수** 버튼 고르기 (왼쪽 **프리셋**을 누르면 한 번에 세팅)
3. **「✓ 확정」** → 오른쪽 아래에 프롬프트 완성
4. **「전체 복사」 / 「전체 .md 다운로드」** 로 가져가거나, **「📌 보관」** 으로 저장

### 2) 자유 빌더(채팅) — 대화로 다듬기
채팅하듯 설명하고(이미지 첨부 가능), 마지막에 **「프롬프트 마감」** 을 누르면 최종본이 나옵니다.

### 3) 보관함 — 저장한 프롬프트 모아보기
**「📌 보관」** 한 프롬프트들이 게시판처럼 쌓입니다. 열어서 복사·다운로드하거나 삭제하세요.

---

## 잘 안 될 때

| 증상 | 해결 |
|---|---|
| 연결이 안 돼요 | 설정에서 **「연결 확인」**. 회사 LiteLLM은 **사내망**에서만 됩니다. |
| 결과가 안 나와요 | 잠시 후 다시 「확정」. 계속 비면 텍스트 모델을 바꿔 보세요(담당자 문의). |
| 화면이 이상해요 | 브라우저에서 **Ctrl + Shift + R** (새로고침) |

## 만든 프롬프트로 이미지 만들기
여기서 만든 프롬프트를 **이미지 스튜디오(codex-image-studio)** 에 붙여 넣어 실제 이미지를 생성합니다.
