# 디자인 개선 기록 — Webflow 디자인 시스템 적용

> 작성: 2026-06-27 · 대상: 프롬프트 빌더 (`static/`) · 참고: Webflow 디자인 시스템(`getdesign.md`)

## 결정 (한 줄)
임시 아이보리/오렌지 테마 → **Webflow 시스템**(흰 캔버스 + near-black 기본 + 5색 크로마틱 카드 + 가중치 ≤600 타이포 + 4px 버튼 + 레이어드 섀도)으로 통일. 한국어 UI라 **Pretendard self-host**.

## 왜 (근거)
1. **신뢰·완성도.** 사내 팀 공용 도구 → "관리자 툴" 인상 대신 "엔지니어링된·차분한" 톤이 적합.
2. **한글 가독성.** UI 대부분이 한국어 → 시스템 폰트는 OS마다 자모 굵기/자간이 달라 완성도 ↓. Pretendard가 한국어 SaaS 사실상 표준.
3. **오프라인.** 사내망·더블클릭 실행 → 외부 CDN 의존 0이 되도록 폰트를 **self-host**(`static/fonts/`).

## 현재 → 개선
| 항목 | As-is | To-be |
|---|---|---|
| 기본색 | 오렌지(#e8732a) 단일 액센트 | **near-black `#080808`** 기본(버튼·선택·강조·제목), 오렌지 제거 |
| 폰트 | 시스템 폰트만 | **Pretendard**(본문) + **Inconsolata**(코드), self-host |
| 라운드 | 혼재 | **버튼/인풋/칩 4px · 카드 8px · 원형 아이콘만 full**, 펄 CTA 금지 |
| 섀도 | 단순 그림자 | **레이어드 멀티스톱 드롭**(featured 카드) + hairline 보더 |
| 강조 | 없음 | **5색 크로마틱**(보라·핑크·파랑·주황·초록)을 프리셋 카드 풀필 시그니처로 |
| 타이포 | 굵기 임의 | **가중치 ≤600 상한**, 디스플레이 음수 자간 |

## 디자인 토큰 (`static/app.css :root`)
- **Surface**: canvas `#fff` · hairline `#d8d8d8` · bot `#fafafa`
- **Ink/Text**: primary/ink `#080808` · ink-strong `#222` · body `#363636` · body-mid `#5a5a5a` · mute `#898989` · mute-soft `#ababab`
- **Chromatic(프리셋/스타일 전용)**: purple `#7a3dff` · pink `#ed52cb` · blue `#3b89ff` · orange `#ff6b00` · green `#00d722`
- **Semantic**: info `#146ef5` · success · warn · error (대비 보정값 사용)
- **Radius**: sm 4px · md 8px · full 9999px
- **Elevation**: `--sh-2`/`--sh-3` 5스톱 레이어드 드롭
- **Font**: `--font-sans`(Pretendard→시스템 한글 폴백) · `--font-mono`(Inconsolata)

## 컴포넌트 규칙
- **버튼**: primary/finalize = near-black 채움(흰 글자, 4px). secondary = 흰 바탕+hairline. ghost = 투명. **펄 금지.**
- **탭**: active = ink 텍스트 + near-black 하단 강조선.
- **옵션 칩(`.opt`)**: 4px·hairline, 선택 시 near-black 채움. (오렌지 대체)
- **카드**(결과·보관함·설정·프리셋): 흰+hairline+8px, featured에 레이어드 섀도.
- **프리셋 사이드바 카드**: **5색 순환 풀필**(위치별 `nth-child`) — Webflow "카테고리 카드" 시그니처. 초록 카드만 가독성 위해 ink 텍스트.
- **스타일 칩**: 앞에 **5색 순환 점**으로 카테고리 색 표시(풀필은 과해서 점으로 절제).
- **코드블록**: near-black 배경 + Inconsolata.

## 폰트 self-host 방법
- `static/fonts/PretendardVariable.woff2`(가변, 400–600), `static/fonts/Inconsolata-400.woff2`.
- `@font-face`로 로드, `/fonts/...` 정적 경로 서빙(StaticFiles). 외부 네트워크 불필요.
- 갱신 시: jsDelivr에서 `pretendard@x / @fontsource/inconsolata` woff2를 `static/fonts/`로 다시 받으면 됨.

## Do / Don't
- **Do** — near-black을 모든 기본 CTA·제목에. 5색은 프리셋 카드 풀필로만. 4px 버튼 / 8px 카드. 가중치 ≤600.
- **Don't** — 5색을 버튼 배경으로 쓰지 말 것. 펄 CTA 금지. 700+ 굵기 금지. 6번째 액센트 추가 금지.

## 범위 / 비고
- **순수 프런트(CSS/폰트)만** 변경 — 백엔드·API·동작 로직 무변경.
- 이모지 아이콘은 오프라인 단순성 위해 유지(후속 lucide 정리는 옵션).
- 디스플레이 크기는 마케팅 페이지(80px)가 아닌 **도구에 맞춰 축소**(제목 18–24px)하되 보이스는 동일.
